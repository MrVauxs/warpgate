    Advanced dialog helper providing multiple input type options as well as user defined buttons. This combines the functionality of buttonDialog as well as dialog

    data object <optional>

    follow the same structure as dialog, buttons follow the same structure

    Properties
    Name Type Attributes Default Description

    buttons Array.<object> <optional>
    		 []

    as buttonDialog
    config object <optional>

    [{title = 'Prompt', defaultButton = 'Ok', options={}}={}] Title of dialog, default button label if no buttons provided,

    Properties
    Name Type Attributes Default Description

    		 'Prompt'
    defaultButton string <optional>
    		 'Ok'
    options object <optional>
    		 {}

    Options passed to the Dialog constructor
    Source:
